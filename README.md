# TaskFlow – Student Task Manager

## Overview

TaskFlow is a MERN Stack web application designed to help students organize, manage, and track their daily tasks efficiently.

The application allows users to create, update, delete, and manage tasks through an intuitive interface while learning modern full-stack development concepts.

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## Planned Features

* Create Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* Task Status Management
* Responsive User Interface
* REST API Integration
